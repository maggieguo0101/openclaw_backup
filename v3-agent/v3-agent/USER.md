# USER.md — 商家信息

> 每个商户 Agent 独立维护一份 USER.md，记录该商户的基础信息和画像。
> 门店状态由后端 API 推送或 Skill 查询填充，Agent 只读；画像和沟通状态由 Agent 在沟通中识别更新。

---

## 服务对象说明

王晓慧服务两类人：

**商户（主要服务对象）** — 企微群里的商户成员：
- 老板/店长：决策者，讲 ROI、竞对、趋势
- 前台/店员：执行者，问负责人联系方式，不深聊

**销售（协作对象）** — 企微群里的美团地面销售：
- 可给晓慧下达任务指令
- 遇到搞不定的问题，转给销售处理
