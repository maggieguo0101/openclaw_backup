## 二、市场数据详情

### 2.1 原油市场

| 指标 | 战前 | 战后 | 变化 |
|------|------|------|------|
| 布伦特原油 | 70美元/桶 | 93美元/桶 | **+33%** |
| 中国原油溢价 | - | 相对布伦特溢价15% | - |
| 亚洲航线原油溢价 | - | 6-10美元/桶 | - |

**背景知识**：霍尔木兹海峡是全球能源供应链咽喉，中东六国石油出口高度依赖该海峡

### 2.2 天然气市场

| 指标 | 战前 | 战前 | 战后 | 变化 |
|------|------|------|------|------|
| 亚洲JKM基准 | 12-13美元/百万英热 | 2月下旬 | 25.4美元 | **+100%** |
| 欧洲TTF基准 | 32欧元(~11美元) | 战前 | 56欧元(~19美元) | **+70%** |
| LNG船租金 | 4万美元/天 | - | 30万美元/天 | **+650%** |

**原因**：全球20%的LNG经过霍尔木兹海峡，卡塔尔90% LNG流向亚洲

### 2.3 柴油市场
- 欧洲高度依赖中东柴油和航空煤油进口
- 霍尔木兹海峡受阻 + 俄罗斯供应缩减
- **柴油裂解价差突破40美元/桶**，创两年多新高

### 2.4 化工品市场
- 经过霍尔木兹海峡运输的金属、化肥、甲醇占全球贸易量**超24%和32%**
- 伊朗是仅次于中国的**第二大甲醇生产国**
- 伊朗是重要的**聚乙烯生产国和出口国**
- 伊朗是中东最大的**尿素出口国**，产能800-900万吨，占全球贸易量**12%**

## 三、股市表现

### 3.1 中国股市
- **"三桶油"涨停**：中石油、中海油、中石化连续两天集体涨停（罕见）
- **A股特点**：对系统性风险不敏感，定价忽视尾部风险
  - 投资者多关注相对收益
  - 机构投资者存在代理人问题
  - 导致市场对系统性风险定价不足

### 3.2 日韩股市
- 前期涨幅较大的日韩股市大幅下跌
- **韩国股市两天暴跌20%**
- 韩国股指因油价暴涨暴跌触发熔断

### 3.3 美股
- **科技股分化**：
  - "花钱的美国大科技巨头"股价不涨（微软不断下跌）
  - 英伟达业绩不错但股价也不涨
  - **受益于AI基建的内存相关股票涨幅大**：三星/SK海力士/海力士年初以来涨**50%**
- **VIX恐慌指数**：温和上涨
- **SKEW焦虑指数**：大幅上升，接近2025年4月关税战"解放日"水平

### 3.4 港股
- 恒生科技指数下跌与伊朗战争无直接关系
- 主要原因：巨头APP流量变化
  - 撒红包后只有豆包存住流量
  - 其他产品流量下降
  - 恒科指数缺乏短期内很强的标的

## 四、航空业影响

### 4.1 航网流量转移
- 中东（迪拜、阿布扎比）因战争不安全，航网流量发生**长期变化**
- 中欧直飞大幅增加
- 伊斯坦布尔和新加坡中转量可能增加
- **中国大陆航空枢纽爆满**：
  - 上海中转长途旅客数量比上周同期**+115%**
  - 北京中转长途旅客数量比上周同期**+110%**
- **中欧直飞票价涨至2万**

### 4.2 航司成本
- 油价上涨增加航司成本
- 燃油附加费上调：
  - 国内航线：800公里以上可能从20元→100-130元→200-220元
  - 国际航线：100%传导，涨幅更大
- 国内航线航司面临与高铁的竞争压力

## 五、金融市场传导机制

### 5.1 传导路径
```
大宗商品上涨 → 通胀压力上升 → 美联储加息预期 → 美债利率上升 → 科技股资本开支压力增大
```

### 5.2 信用环境
- 美国信用环境在2025年2月初开始略微紧缩
- 支持科技股的资本开支依赖体外影子融资
- 私募信贷参与其中

### 5.3 私募信贷风险事件
| 机构 | 事件 |
|------|------|
| 蓝鸮资本（Blue Owl Capital） | 暂停旗下一只基金赎回 |
| 贝莱德 | 开始限制旗下私募信贷基金赎回 |
| 美国地区银行股 | 2月初以来大幅下跌 |

- 油价飙涨 → 美债利率上升、美元汇率反弹 → 信用环境脆弱性加剧
- **市场担忧私募信贷爆雷**：资产池面临流动性冲击
- 目前未引发大规模违约，但存在潜在风险

## 六、黄金分析

### 6.1 黄金属性
- **货币备胎**：对冲美元信用风险

### 6.2 战争影响的不确定性
| 情景 | 对黄金影响 |
|------|-----------|
| 美国被拖入长期战争泥潭 + 美联储稳住利率和美债 | **有利** |
| 美国顺利重整中东秩序，战争快速结束 | 影响不明 |

### 6.3 当前操作
- 因黄金单位波动增加
- 按风险配置原则，**仓位下降**
- **目前难以确定投资方向**

## 七、投资策略建议

### 7.1 核心观点
- **能源是今年的核心矛盾点**
- 围绕能源相关的生产、化工材料、设备制造等产业可能有大机会
- 油价波动大幅上升，是推动其他行情演绎的**核心**
- **不意味着油价一定上涨，但抓住能源才能把握其他投资机会**

### 7.2 具体方向
- **VLCC油运**：原本过剩出清，委内瑞拉事件+伊朗紧张局势推动价格上涨
- 全球供应链和交通运输行业：提前布局，在供需偏紧平衡时遇到地缘冲击能获得收益

### 7.3 策略转变
- 从**carry策略**过渡到**动量策略**
- 将主观发现的主要矛盾**规范化、范式化**
- 提高投资效率和纪律性

### 7.4 反脆弱体系
- **第一步**：学会洞察脆弱点
- **第二步**：寻找"凸性资产"——位于生态链中紧平衡的节点
- 让不确定性的爆发对自己有利，**无需预测爆发时间**
