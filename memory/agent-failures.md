## 案例记录

### 2026-03-23/24 晨光电机（920011）打新数据抓取失败
- **现象**：无法获取 920011 发行参数，分析任务被迫中断，等待老板手动提供数据
- **根因分类**：Harness（网络）
- **具体原因**：沙箱 proxy（127.0.0.1:8118）对 bse.cn 返回 403 Forbidden；浏览器工具 timeout；catclaw-search RPM 限制触发
- **影响**：任务失败，无法自主完成打新分析
- **改进措施**：
  1. 遇到 bse.cn 数据需求，优先告知老板需要手动提供参数，不要反复重试浪费时间
  2. 尝试腾讯行情 API 作为备用（只有行情，无发行参数）
  3. 长期：调研是否有可访问的北交所数据源
- **状态**：已绕过（人工提供数据）
