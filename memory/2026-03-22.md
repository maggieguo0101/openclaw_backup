## 手抄报 Skill 创建

### 背景
老板要求给小学生做元宵节手抄报，调研了手抄报标准结构后发现现有技能不覆盖该场景：
- `canvas-design`：成人艺术风格，不适合儿童
- `pptx/docx`：文档格式，不符合图文混排+装饰需求

### 创建了新 Skill：`shou-chao-bao`
- 安装路径：`/root/.openclaw/skills/shou-chao-bao/SKILL.md`
- 也备份于：`/root/.openclaw/workspace/skills/shou-chao-bao/SKILL.md`

### 手抄报标准结构（调研结论）
1. **报头区**：顶部大标题 + 副标题 + 主题装饰
2. **内容框 2-5 个**：彩色标题条 + 文字内容
3. **绘画/手写区**：预留空白，横线供孩子填写
4. **装饰元素**：角花、花边、卡通简笔画
5. **底部署名区**：姓名/班级/日期

### 关键技术约束（踩坑记录）
- **截图卡死根因**：HTML 含 Google Fonts `@import url(...)` 外链，沙箱网络不通导致 playwright 等待超时
- **解决方案**：删除所有外部资源，用 `wait_until='domcontentloaded'`（不用 `networkidle`）
- **playwright 截图固定写法**：
  ```python
  browser = p.chromium.launch(
      executable_path='/usr/bin/chromium-browser',
      args=['--no-sandbox','--disable-dev-shm-usage','--disable-web-security']
  )
  page.goto(url, wait_until='domcontentloaded', timeout=10000)
  ```
- **字体**：`'FangSong','SimSun','STSong','Noto Serif CJK SC',serif`
