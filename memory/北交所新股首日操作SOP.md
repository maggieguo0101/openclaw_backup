# 北交所新股首日操作SOP v2.0

> 最后更新：2026-03-18，族兴新材复盘后迭代
> 核心逻辑：用换手率判断资金意愿，换手率斜率坏了才是趋势破坏，价格短暂回调不是信号。

## 阶梯换手率门槛

| 时间点 | 换手率门槛 | 不达标动作 |
|---|---|---|
| 9:25 集合竞价 | 7% | 开盘全部卖出 |
| 9:35 | 10% | **立刻全部卖出**（挂比当前价略高的限价单） |
| 9:45 | 30% | **立刻全部卖出** |
| 10:00 | 45% | **立刻全部卖出** |

## 达标后的操作

- **每个时间点达标 → 继续持有，等下一个时间点**
- 10:00 达标（换手≥45%）→ 进入"涨停博弈"阶段
  - 封住涨停 → 等第二个涨停再决策
  - 封不住（开板）→ **立刻全部卖出，不犹豫**

## ⚠️ 铁律：价格回调不是卖出信号

> **价格短暂回调 ≠ 趋势破坏 ≠ 卖出信号**
> 只要换手率达标，价格回调是洗盘，不是趋势破坏。
> **唯一触发卖出的信号：换手率不达标，或涨停开板。**
> 情绪驱动的卖出 = 规则失效，复盘必追责。
