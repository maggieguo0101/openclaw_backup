一、用户生命周期总览
🆕 潜客
目标：拉新
指标：新客数
🔥 活跃期
目标：提频
指标：N转N+1复购率
⚠️ 衰退期
目标：留存
指标：流失率
🚨 流失期
目标：召回
指标：召回率
二、潜客期：裂变拉新 拉新
玩法1：基础裂变活动
老拉新砍价/砍券，客态用户帮砍同时注册为新用户
玩法2：裂变 + 任务制
帮砍后引导客态做简单任务（浏览POI、浏览详情页）→ 翻倍红包
核心：不是为自己获利，而是帮朋友砍更多 → 成就感驱动
玩法3：裂变 + 答题制
帮砍后答题获得翻倍红包。题目围绕业务认知设计
目的：让流量不只帮砍就走，而是形成品牌认知
⭐ 玩法4：秒杀大额券（锁定新客）
① 0.01元秒杀50元券 → 抓摇摆用户（几乎无成本）
② 10元秒杀100元券 → 锁定有需求新客（沉没成本）
买券 = 付出成本 → 后续转化概率大幅提升
玩法5：券金额膨胀
领券后未使用 → 金额自动膨胀刺激转化
新客无门槛膨胀 / 老客需拉新才能膨胀
玩法6：做任务送现金
基础参与得1元 → 邀1新人得3元 → 邀3新人得5元 → 好友下单得20元
满10元提现，对标拼多多/同程模式
三、活跃期：提频促忠诚 核心
"提频是结果，可以通过优惠来培养用户的忠诚。用户是先有消费，消费多了才慢慢转为忠诚。"
"忠诚度培养的关键：每次消费中增加用户的沉没成本"
⭐ 玩法1：积分主流程抵现
100积分 = 1元，下单页直接展示「您有XX积分可抵X元」
让用户感知积分值钱 → 积分就是沉没成本 → 不舍得离开
参考：京东
⭐ 玩法2：任务制下单返现
参与下单任务 → 额外获得返现补贴
让用户感知「订得越多越便宜」→ 对平台产生依赖
参考：艺龙
玩法3：会员分层红包
不同等级会员 → 不同额度专享券
前置页投放会员专享券，强化等级权益感知
玩法4：会员体验月卡
券 + 权益打包销售（如延时退房、折扣等）
低级会员购买 → 体验高级权益 → 促成长升级
参考：拼多多省钱月卡
玩法5：下单抽免单
3元参与抽奖 → 100%中奖 → 最高免单
每200人开一次奖：1等免单 / 2等红包 / 3等积分
花钱买券 → 付出成本 → 记住使用概率提高
参考：同程
玩法6：商家升房券/早餐券
商家出资提供增值服务券 → 平台给流量
低价享高价服务 → 由奢入俭难 → 消费升级
四、衰退期：留存防流失 留存
"需要让用户明白，如果离开平台，会承受一定的损失"
⭐ 玩法1：直接发放返现（非券！）
Push/短信直接往账户发返现金额（不是券）
提现条件：拉新注册 / 拉新下单 / 自己下单
本质：换一种玩法的老拉新
玩法2：已享优惠提醒（年度账单式）
Push已省多少钱 → 已消费多少 → 积分余额（折算成钱）→ 已享服务
通过内容唤起用户对平台的认知和情感连接
玩法3：红包临期提醒
券快过期时Push提醒 → 制造紧迫感促转化
玩法4：红包延期（付费）
大额券临期 → 允许延期1次（需积分/拉新）
筛选用户：参与延期的 = 后续高转化概率
玩法5：券金额膨胀
无大额券的用户 → 参与膨胀活动 → 付出成本获大券
玩法6：免单抽奖（降门槛）
衰退期用户 0.1元（vs活跃期3元）参与免单抽奖
免费房由平台出，选本地周边度假型
五、核心方法论提炼
原则解释应用
沉没成本用户付出越多越不舍得走积分抵现、买券、任务返现
损失厌恶失去比获得更痛苦临期提醒、已享优惠账单、进度条
用户分层不同阶段不同策略潜客重认知、活跃重频次、衰退重挽留
任务制行为引导 + 即时反馈下单返现、拉新返现、做任务送钱
社交裂变利用社交关系降成本帮砍、邀新、好友下单返现
📌 出行场景落地建议：

① 活跃期：连续下单阶梯奖励 + 积分主流程抵现 + 通勤包
② 衰退期：直接发返现（非券）+ 已省钱账单Push + 临期提醒
③ 潜客期：秒杀大额券（0.01元/10元）锁定新客第一单

最关键的一句话：提频是结果，沉没成本是手段。
