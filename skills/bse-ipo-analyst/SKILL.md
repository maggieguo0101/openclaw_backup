# 北交所新股申购分析 Skill

## 核心工作流

### 1. 查看近期申购列表
```bash
python3 scripts/bse_analyze.py pending    # 未来7天待申购
python3 scripts/bse_analyze.py list       # 所有发行中新股
```

### 2. 分析指定新股
```bash
python3 scripts/bse_analyze.py analyze 普昂    # 按名称搜索
python3 scripts/bse_analyze.py analyze 920069  # 按申购代码
```

输出 JSON，包含：发行价、PE、顶格资金、正股门槛估算、三档年化收益率。

### 2.5 核实官方顶格数据（重要）

脚本估算顶格上限基于发行量×5%，**误差可能较大**，需补充搜索确认：
```
搜索：[股票名] 顶格申购 万股 发行公告
搜索：[股票名] 920XXX 网上发行 申购上限
```
重点核实：① 网上发行量 ② 单账户申购上限股数 ③ 顶格所需资金

### 3. 组织分析报告

拿到 JSON 后，按以下结构输出报告：

```
📊 [股票名]（[申购代码]）申购分析

🏭 公司基本面（搜索补充：主营、营收、净利润趋势、风险点）

💰 发行参数
- 发行价 / PE / 定价方式
- 顶格资金 / 正股门槛 / 1签缴款金额

🎯 不同资金量获配结构
| 资金量 | 可申购股数 | 预期获配 |
（正股靠资金排名，碎股靠时间优先）

📐 收益率测算（三档：悲观/中性/乐观）
| 情景 | 开盘价 | 每签收益 | 年化收益率 |

⚡ 申购建议
- 推荐券商 + 最早操作时间（见 references/rules.md）
- 是否需要拼手速（门槛附近资金必须拼）
- 首日操作策略（换手率规则）
```

### 4. 补充基本面数据

脚本只提供发行参数，基本面需搜索：
```
搜索：[股票名] [代码] 营收 净利润 2024 2025
搜索：[股票名] 主营业务 行业 竞争格局
```

关注：营收增速趋势、毛利率、核心风险点（降价压力/客户集中度/出口风险等）。

### 5. 预计开盘价计算

```
行业PE可用搜索获取，或用经验倍数：
- 保守：发行价 × 1.5
- 中性：发行价 × 2.0  
- 乐观：发行价 × 2.5

精算：(行业PE / 发行PE) × 发行价
```

## 关键规则（无需每次查 references）

- **1签 = 100股**，缴款 = 发行价 × 100
- **不冻结资金**，中签后才缴款
- **正股靠资金**，碎股靠时间（同档资金越早越好）
- **银河/中银 8:30** 是目前最早的券商
- 上市首日：换手 < 20% → 立刻清仓
