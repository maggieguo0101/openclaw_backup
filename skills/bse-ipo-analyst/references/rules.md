# 北交所打新规则参考

## 基本规则

| 项目 | 规则 |
|------|------|
| 1签 | = 100股 |
| 申购时间 | 交易日 9:15~11:30、13:00~15:00 |
| 资金冻结 | **不冻结资金**，中签后才缴款 |
| 缴款金额 | 发行价 × 100股 |
| 中签结果 | 申购日次日公布 |
| 缴款日 | 中签公布后约T+4 |

## 申购规则说明

### 正股 vs 碎股
- **正股**：整签（100股整数倍）→ 按**资金量**从高到低分配
- **碎股**：不足100股的余量 → 先按**资金量**，同档位内按**时间优先**分配
- 顶格申购 = 发行量上限对应资金，确保正股最大化，同时有资格竞争碎股

### 获配结构（X+Y）
- X = 正股签数（每签100股），Y = 碎股（< 100股）
- 例：顶格申购，资金1389万，预计获配"3+1"= 300股正股 + 约50股碎股

### 定价方式
- **直接定价**（enquiryType=3）：价格锁定，申购确定性最高
- **询价**（enquiryType=1）：机构定价，波动可控
- **竞价**（enquiryType=2）：较少见

## 年化收益率计算公式

```
每签收益 = (预计开盘价 - 发行价) × 100
年化收益率 = 每签收益 / (顶格资金元) × (365 / 资金占用天数) × 100%
资金占用天数 ≈ 4天（申购日到缴款日）
```

### 预计开盘价估算方法
北交所直接定价新股发行PE通常卡在14~15x以内，而二级市场同类公司PE往往更高：
- **保守估计**：发行价 × 1.5（约±1x估值修复）
- **中性估计**：发行价 × 2.0（行业PE还原）
- **乐观估计**：发行价 × 2.5（市场情绪好）

可用搜索查行业平均PE后精算：`(行业PE / 发行PE) × 发行价`

## 申购策略总结

| 资金情况 | 建议 |
|---------|------|
| < 正股门槛 | 放弃，性价比极低 |
| 正股门槛 ~ 顶格×80% | 全额申购，拼手速（尽量用8:30券商） |
| 顶格×80% ~ 顶格 | 补至顶格，越早越好 |
| 顶格 | 越早提交越好，碎股靠时间优先 |

## 上市首日操作规则（北交所）

| 换手率 | 操作 |
|--------|------|
| > 40% | 可以等冲高，分批出 |
| 20~40% | 先走一半，留仓观察 |
| < 20% | **立刻全部卖出，不犹豫** |

> 下午流动性极差，原则上上午解决。
