# 北交所打新全流程 Skill

## 脚本位置
```
skills/bse-ipo-full/scripts/bse_pipeline.py
```

## 五个流程入口

### 流程1：排队预估（注册批文监控）
```bash
python3 scripts/bse_pipeline.py pipeline
```
- 查询已注册生效但未上市的股票，预估申购日/上市日
- 查询已过会但未注册的股票，预估注册时间
- 输出全流程时间线参考（基于2024-2025历史数据）

**时间线基准（2025年最新）：**
| 阶段 | 平均天数 |
|------|---------|
| 注册生效 → 上市 | 31天 |
| 注册生效 → 申购 | ~7天 |
| 过会 → 注册生效 | ~63天（48+15） |
| 全程（受理→上市） | ~339天 |

### 流程2：申购/上市日历
```bash
python3 scripts/bse_pipeline.py calendar
```
- 列出未来30天所有申购日、中签日、上市日事件
- 距今≤3天的事件标记 ⚡ 警报

### 流程3：申购分析
```bash
python3 scripts/bse_pipeline.py analyze 普昂
python3 scripts/bse_pipeline.py analyze 920069
```
输出：
- 发行参数（价格/PE/定价方式/发行量）
- 资金需求（1签缴款 / 顶格估算）⚠️ 需核实官方公告
- 多账户资金分配建议（5万/20万/50万/顶格）
- 三档收益率测算（悲观/中性/乐观）
- 申购操作建议（券商选择/时间窗口）

**补充核实顶格上限（脚本为估算）：**
```
搜索：[股票名] 顶格申购 万股 发行公告
```

### 流程4：上市日分析
```bash
python3 scripts/bse_pipeline.py listing 920069
```
- 获取实时行情（腾讯API）
- 开盘价预测三档
- 首日卖出SOP提示

### 流程5：复盘记录
```bash
python3 scripts/bse_pipeline.py review 920069
```
- 获取当前行情快照
- 生成标准复盘模板
- 填写后追加到 `memory/ipo-strategy.md`

## 首日卖出SOP（核心规则）

基于已积累样本（新恒泰+普昂医疗）：

| 信号 | 判断 | 动作 |
|------|------|------|
| 开盘换手率 < 20% | 开盘即高点型 | 立刻全卖 |
| 开盘换手率 20-40% + 价格上涨 | 持续拉升型 | 观察，不急卖 |
| 量增价升（换手高+价格新高） | 主力进场 | 继续持有 |
| **量增价滞/价跌（量价背离）** | **出货信号** | **立刻卖** |
| 换手率 > 50% + 价格停止上涨 | 高位分歧 | 分批卖 |

**⚠️ 关键区分：**
- 量增 ≠ 卖出信号
- 卖出信号 = **价格不涨了，但换手还在飙**

## 关键参数参考

### 券商最早申购时间

|------|---------|

| 中银证券 | **8:30** ⭐ |
| 国金证券 | 8:35 |
| 招商/华泰/东方/平安 | 8:45 |
| 国信证券 | 8:50 |

### 申购规则
- 1签 = 100股，缴款 = 发行价 × 100
- **不冻结资金**，中签后才缴款
- 正股靠资金量（越多越好）
- 碎股靠时间优先（同档资金越早越好）

- 每天上午10点：运行 `calendar` + `pipeline`，有申购日则推提醒
- 上市日当天：运行 `listing`，推行情快照
- 用户主动触发复盘：运行 `review`
