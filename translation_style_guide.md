# 翻译风格指南 — 晓飞风格（Steve Pavlina 系列）

## 确认时间
2026-03-11，老板确认后续翻译统一使用此风格

## 风格DNA

### 1. 句式结构
- **保持原文长句结构**，不刻意拆分短句
- 用中文化处理让长句读起来流畅自然
- 逐句忠实对应，不删不漏

### 2. 用词偏好
- **大量使用四字词/成语**增强节奏感（轻松自然、深入领略、慷慨呈现、无比亲密、痛快应允）
- 动词要生动有力（"激起情欲" > "性唤起"，"滋滋作响" > "嗡嗡作响"）
- 适度添加修饰语增强画面感（"绒毯" > "毯子"，"温柔包裹" > "包裹"）

### 3. 核心概念译法
- vibrational → 同频共振（意译，不硬译"振动/振频"）
- reality → 现实/现实世界（不用"实相"）
- exploration → 探索尝试
- embodied → 具体可感/更加具体化
- deflection → 偏离回避
- intimacy → 亲密体验
- 30-day challenge → 30天挑战
- blanket of love → 爱的绒毯

### 4. 人名处理
- 保留英文原名（Rachelle 不译为"瑞雪儿"）
- 作者署名格式：Steve Pavlina（史蒂夫·帕弗利纳）

### 5. 语气温度
- 有温度、带感受，但不过度煽情
- 比直译更饱满丰富
- 抽象概念做意译而非直译

### 6. 标题格式
- 保留完整系列名 + 期数
- 格式：**100天性爱试验：第X天**
